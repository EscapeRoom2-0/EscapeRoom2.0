========================================English===============================================
Hello Programmer, this is the EscapeRoom 2.0 Project for ETEC Pedro Ferreira Alves.

To make this program functional, follow these steps:

1. Run composer install to install all dependencies.

2. Copy the environment example file by running:

cp .env.example .env

3. Finally, apply the database migrations and seeders with:

php artisan migrate --seed

Additionally, the source code has been modified for specific functionality, 
allowing the game to communicate between different computers or browsers. 
You will need to make the following modification as well:

Navigate to: (application directory)/vendor/laravel/framework/src/Illuminate/Session/Store.php

At line 672, replace:

return Str::random(40);

with:

return 'session';

Now, you can enjoy the game/code! :)
